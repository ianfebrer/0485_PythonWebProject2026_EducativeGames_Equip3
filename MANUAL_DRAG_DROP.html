Manual per connectar Drag & Drop al rànquing

Quan el joc drag_drop estiga acabat, aquests són els passos que has de seguir perquè guarde puntuacions i aparega correctament en el rànquing.

1. Crear la ruta del joc

En routes/game_routes.py afegeix aquesta ruta:

@game_bp.route('/drag-drop')
def play_drag_drop():
    return render_template('games/drag_drop.html')

2. Crear l'endpoint per guardar la puntuació

També en routes/game_routes.py afegeix:

@game_bp.route('/api/guardar-resultat-drag-drop', methods=['POST'])
def api_guardar_resultat_drag_drop():
    dades = request.get_json() or {}
    punts = dades.get('punts', 0)

    try:
        punts = int(punts)
    except (TypeError, ValueError):
        punts = 0

    missatge = f'Has aconseguit {punts} punts!'

    if session.get('username'):
        username = session.get('username')
        millor_puntuacio = score_storage.update_user_score(username, 'drag_drop', punts)
        guardat = True
        missatge += f' Millor puntuacio guardada: {millor_puntuacio}.'
    else:
        guardat = False
        missatge += ' Inicia sessio per guardar els teus resultats!'

    return jsonify({
        'punts': punts,
        'guardat': guardat,
        'missatge': missatge
    })

3. Enviar la puntuació des del JavaScript del joc

Quan el jugador acabe la partida, fes un fetch cap al backend:

fetch('/api/guardar-resultat-drag-drop', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        punts: puntuacioFinal
    })
})
.then(response => response.json())
.then(data => {
    console.log(data.missatge);
});
Important: puntuacioFinal ha de ser la variable final del vostre joc.

4. Connectar l'enllaç del joc en la pàgina principal

En templates/index.html, canvia el botó de Drag & Drop:

<a href="{{ url_for('game.play_drag_drop') }}" class="btn btn-accent">Jugar ara</a>

5. Comprovar que scores.json s'actualitza

Després de jugar una partida amb un usuari loguejat, revisa data/scores.json. Hauria de quedar una estructura així:

{
    "username": "nom_usuari",
    "scores": {
        "teclado": 0,
        "raton": 0,
        "drag_drop": 120
    }
}

6. Revisar el rànquing

Obri /ranking i comprova que la taula de Drag & Drop mostra la nova puntuació.

7. Opcional: llevar el text de plantilla

En templates/ranking.html, si ja no vols l'avís que el joc està pendent, elimina eixe text.

Coses que no has de tocar

  • models/score_storage.py
  • data/results.json
  • data/scores.json, excepte per comprovar que es guarda bé

Resum ràpid

  1. Crear la ruta del joc.
  2. Crear l'endpoint per guardar punts.
  3. Fer el fetch des del JS.
  4. Canviar l'enllaç del botó.
  5. Comprovar scores.json.
  6. Revisar /ranking.
